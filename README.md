# python-bank-management-system
Python Bank Management System is a menu-driven console application that allows users to create bank accounts, deposit and withdraw money, check balances, and view all accounts. It uses file handling to permanently store account data.

A simple command-line based Bank Management System developed using Python. This project allows users to manage bank accounts, perform transactions, and store data permanently using file handling.
